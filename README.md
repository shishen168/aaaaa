# 神域短信平台

专业的短信服务解决方案

## 功能特点

- 支持单发/群发短信
- 短信模板管理 
- 联系人管理
- 黑名单管理
- 定时发送
- 发送历史记录
- 多语言支持
- 响应式设计

## 技术栈

- React 18
- TypeScript
- Tailwind CSS
- Vite
- React Router
- i18next
- Lucide Icons

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 部署

本项目使用 Netlify 进行自动部署。
